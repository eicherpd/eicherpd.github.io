Intrusive Kinetic Design

The purpose of this project was to use motion design to personify the qualities of an adjective. The adjective in this instance was 'intrusive.' The video displays a bouncing object morphing until it changes into an object of penetrating the 'wall' form and intruding into the next field. The project was done in Adobe After Effects for Kinetic Design 1.
